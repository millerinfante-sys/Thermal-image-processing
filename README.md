# Thermal-image-processing
Este script es un modelo para analizar imágenes térmicas, detectar posibles "puntos calientes" o anormalidades térmicas, y clasificarlas utilizando un modelo de Machine Learning. El sistema permite una interacción con el usuario para validar las predicciones del modelo y reentrenarlo. Es una herramienta útil para el monitoreo de equipos eléctricos
